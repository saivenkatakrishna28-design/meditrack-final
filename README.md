# MediTrack – Smart Medicine Reminder & Basic Health Tracker

MediTrack is a health-tech prototype built to help users manage medicine reminders, monitor daily health metrics, and store emergency contact information in one place through a modern and easy-to-use interface.

## Problem Statement

Many people forget to take medicines on time and do not regularly track simple health indicators such as blood pressure, sugar level, water intake, sleep, and weight. This creates a gap in daily health monitoring and reduces emergency preparedness.

## Solution

MediTrack provides a lightweight digital solution that combines:

- medicine scheduling
- health data logging
- emergency contact storage
- dashboard-based monitoring
- downloadable summary reporting

## Key Features

- Add and manage medicine reminders
- Store dosage, time, frequency, and notes
- Log daily health metrics:
  - weight
  - blood pressure
  - sugar level
  - water intake
  - sleep hours
- View a live dashboard with:
  - total medicines
  - today's water intake
  - latest sleep data
  - health score
- Save emergency contact details
- Download a text-based health summary report
- Browser notification support for reminders
- Demo data button for quick testing and presentation

## Why This Project Matters

- Improves medication adherence
- Encourages consistent daily monitoring
- Helps users stay better prepared during emergencies
- Can be extended into a real scalable health-tech product
